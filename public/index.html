Introducción

Bienvenido a la documentación de nuestra API. Aquí encontrarás toda la información necesaria para integrar nuestros servicios en tu aplicación.

Autenticación

Nuestra API utiliza autenticación basada en tokens. Sigue estos pasos para obtener y utilizar tu token de acceso.

  1. Regístrate para obtener una cuenta de desarrollador.
  2. Genera tu token de API en el panel de control.
  3. Incluye el token en el encabezado de tus solicitudes.

Endpoints

GET /api/v1/cat

Obtiene información sobre gatos.

Parámetros

Esta petición no requiere parámetros.

Respuesta

{
  "id": "c-001",
  "name": "Whiskers",
  "breed": "Siamese",
  "age": 3
}

GET /api/v1/dog

Obtiene información sobre perros.

Parámetros

Esta petición no requiere parámetros.

Respuesta

{
  "id": "d-001",
  "name": "Buddy",
  "breed": "Labrador",
  "age": 5
}

Ejemplos de Uso

Aquí tienes un ejemplo de cómo hacer una solicitud a nuestra API usando JavaScript:

fetch('https://api.ejemplo.com/v1/cat', {
  headers: {
    'Authorization': 'Bearer TU_TOKEN_AQUI'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
