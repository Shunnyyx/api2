Introducción

Bienvenido a la documentación de la API. Aquí podrás encontrar información sobre cómo utilizar la API.

Autenticación

Para acceder a la API, necesitas autenticarte usando una clave de API. Aquí te explicamos cómo obtenerla.

Endpoints

GET /api/v1/cat

Descripción: Obtiene información sobre gatos.

Parámetros: Ninguno

Respuesta:

{
  "data": "info sobre gatos"
}

GET /api/v1/dog

Descripción: Obtiene información sobre perros.

Parámetros: Ninguno

Respuesta:

{
  "data": "info sobre perros"
}

Guía Rápida

Para empezar rápidamente, aquí tienes algunos ejemplos de cómo hacer solicitudes a la API.

fetch('/api/v1/resource')
  .then(response => response.json())
  .then(data => console.log(data));
